# My Game Hub

A futuristic browser-based game launcher with neon UI and interactive game cards. Built with pure HTML, CSS, and JavaScript, it features a login & boot screen, XP leveling, daily rewards, and 3D card animations.

## Features

- Animated boot screen and login sequence  
- XP system with daily rewards  
- Interactive flipping game cards with 3D hover tilt  
- Play popular web games directly from the hub  
- Favorites and score tracking using localStorage  

## How to Run

1. Clone or download this repository  
2. Open `index.html` in your web browser  
3. Enjoy the hub and play games!

## Tech Stack

- HTML  
- CSS  
- JavaScript (Vanilla)  

## Notes

- Login is client-side only; not secure for real accounts  
- Works on desktop and mobile browsers
